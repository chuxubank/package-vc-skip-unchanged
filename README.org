#+title: package-vc-skip-unchanged

Skip unnecessary work in =package-vc-upgrade-all= when a Git package's local
=HEAD= already matches its upstream commit.

The package fetches repositories with bounded concurrency, compares commit
hashes, and calls the native =package-vc-upgrade= only for packages that have
changed.  Non-Git packages and repositories that cannot be checked fall back
to the native behavior.

* Usage

Install directly from GitHub with Emacs 30:

#+begin_src emacs-lisp
(package-vc-install
 "https://github.com/chuxubank/package-vc-skip-unchanged")
#+end_src

Then enable the global mode:

#+begin_src emacs-lisp
(use-package package-vc-skip-unchanged
  :config
  (package-vc-skip-unchanged-mode 1))
#+end_src

The default concurrency is 6.  Customize it when needed:

#+begin_src emacs-lisp
(setq package-vc-skip-unchanged-max-concurrency 4)
#+end_src

Loading the package alone does not modify =package-vc-upgrade-all=; the global
minor mode explicitly enables and disables the override.

* Testing
