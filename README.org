#+title: package-vc-skip-unchanged

Skip unnecessary work in =package-vc-upgrade=, =package-vc-upgrade-all=, and
=package-upgrade-all= when a Git package's local =HEAD= already matches its
upstream commit.

Bulk VC upgrades fetch repositories with bounded concurrency, compare commit
hashes, and call the native =package-vc-upgrade= only for packages that have
changed.  Direct upgrades perform the same check synchronously.  Non-Git
packages and repositories that cannot be checked fall back to the native
behavior.

* Usage

Install directly from GitHub with Emacs 30:

#+begin_src emacs-lisp
(package-vc-install
 "https://github.com/chuxubank/package-vc-skip-unchanged")
#+end_src

Then enable the global mode:

#+begin_src emacs-lisp
(use-package package-vc-skip-unchanged
  :config
  (package-vc-skip-unchanged-mode 1))
#+end_src

The default concurrency is 6.  Customize it when needed:

#+begin_src emacs-lisp
(setq package-vc-skip-unchanged-max-concurrency 4)
#+end_src

Loading the package alone does not modify package functions; the global minor
mode explicitly enables and disables the upgrade advice and bulk override.
